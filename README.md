# Inventory Management System (Python)

A simple console-based inventory management system built using Python.

## Features

- Add new products
- View a specific product
- View all products
- Update product quantity
- Remove products
- Input validation for:
  - Product names
  - Quantity
  - Price
- Case-insensitive product handling

## Technologies Used

- Python 3
- Dictionaries
- Functions
- Loops
- Input validation (try/except)
